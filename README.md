# Simple Calculator

## Description
This project is a simple console-based calculator developed using Java.  
The calculator performs basic arithmetic operations based on user input.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User input handling
- Division by zero validation

## Technologies Used
- Java
- Eclipse IDE

## Concepts Used
- Scanner Class
- Switch Case
- Conditional Statements
- Arithmetic Operators
- Console Input/Output

## Project Structure

Calculator.java

## How to Run

1. Open the project in Eclipse IDE
2. Run the Calculator.java file
3. Enter two numbers
4. Choose the operation
5. View the result in console
